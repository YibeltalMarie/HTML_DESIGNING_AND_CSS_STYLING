Flexbox Navigation Bar Demo

About This Navigation Bar

This navigation bar is built using CSS Flexbox. The logo is on the left and the menu links are on the right, with space between them.

Resize your browser window to see the responsive behavior. On mobile screens, the navigation items will stack vertically.

How Flexbox Works

Flexbox is a CSS layout module that makes it easy to create flexible and responsive layouts. The navigation bar above uses:

display: flex - Makes the navbar a flex container

justify-content: space-between - Puts logo on left, menu on right

align-items: center - Vertically centers items in navbar
