Position Property Demonstration

Understanding CSS Position

The CSS position property controls how elements are positioned on a web page. It's essential for creating complex layouts and UI components.

This page demonstrates three different position values:

  • Fixed position - Navigation bar at top (always visible)
  • Sticky position - Header that sticks after scrolling
  • Fixed position - "Back to Top" button at bottom-right

Page Content

Scroll down to see the sticky header become fixed and to see the "Back to Top" button in action.

Fixed Navigation

The blue navigation bar at the top of the page uses position: fixed. It stays in the same place even when you scroll.

Sticky Header

The gray header below the navigation uses position: sticky. It will stick to the top of the page once you scroll past it.
