Active Selectors Demonstration

This menu demonstrates all the required selectors:

  • first-child: First item is bold
  • :hover: Links change color on hover
  • Attribute selector: Active page has different background
  • Child selector (>): Direct children of nav have borders

Selector Concepts Explained

Child Selector (nav > ul)

Selects only direct children. Notice borders on main menu items only, not submenu items.

nav > ul > li { border: 2px solid #e74c3c; }
first-child Pseudo-class

Selects the first child element. The "Home" link is bold.

.menu > li:first-child { font-weight: bold; }
:hover Pseudo-class

Applied when user hovers over an element. Try hovering over menu items!

.nav-link:hover { color: #2ecc71; }
Attribute Selectors

Selects elements based on attributes. Active page (Contact Us) has orange background.

a[aria-current="page"] { background-color: #f39c12; }

Try This Experiment

Hover over different menu items to see hover effects. Notice:

  1. Only direct children of nav have borders (not submenu items)
  2. First item ("Home") is bold
  3. "Contact Us" has different background (it's the active page)
  4. All links change color on hover

Additional Selector Examples

/* Attribute contains selector */ a[href*="#page"] { color: purple; } /* Attribute starts with selector */ a[href^="#"] { text-decoration: none; } /* Attribute ends with selector */ a[href$="dev"] { font-style: italic; } /* Multiple attribute selector */ a[data-page][href] { padding: 5px 10px; }
